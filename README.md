# 📚 图书管理系统

![Spring Boot](https://img.shields.io/badge/Spring%20Boot-4.0.0-brightgreen)
![Java](https://img.shields.io/badge/Java-17-orange)
![MySQL](https://img.shields.io/badge/MySQL-8.0-blue)
![License](https://img.shields.io/badge/License-MIT-yellow)

一个基于Spring Boot和原生JavaScript开发的全栈图书管理系统，适合作为Java后端入门项目展示。

## ✨ 项目亮点

- **后端**：Spring Boot + Spring Data JPA + MySQL
- **前端**：纯HTML/CSS/JavaScript + Axios
- **架构**：清晰的MVC分层设计
- **功能**：完整的CRUD操作 + 模糊搜索

## 🛠️ 技术栈

### 后端
- Spring Boot 4.0.0
- Spring Data JPA
- MySQL 8.0
- Maven

### 前端
- HTML5 + CSS3
- JavaScript (ES6)

### 工具
- VSCode
- JDK 17
- MySQL Workbench / 命令行

##  功能

### 1. 主界面 - 图书列表
![主界面](/booksystem%20screenshot.png)
*图书管理系统主界面，展示所有图书信息*

### 2. 添加图书
*通过表单添加新图书，支持书名、作者、ISBN和价格*

### 3. 搜索功能
*根据书名或作者进行模糊搜索*

### 4. 删除图书
*删除图书记录并实时刷新列表*
